# Introduction

A Simple Grocery Shopping ToDo App built using the MVC Architecture, we have also implemented "authorization" so folx can sign up, customize & personalize the app.

---


---

# Objectives

- It's a beginner level app created to understand how MVC concept and logins are added

---

# Who is this for? 

- It's for beginners & intermediates with little more experience, to help understand the various aspects of building a node app with some complex features

---

# Packages/Dependencies used 

bcrypt, connect-mongo, dotenv, ejs, express, express-flash, express-session, mongodb, mongoose, morgan, nodemon, passport, passport-local, validator


---

# Optmizations

Add export functionality so users can share lists via text and email.
